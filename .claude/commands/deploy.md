# Deploy Command

Deploy the agency portfolio website to production.

## Steps:
1. Verify build process works locally
2. Check environment variables
3. Run tests and linting
4. Build for production
5. Deploy to Supabase/Heroku
6. Verify deployment success

## Usage:
Type `/deploy` to start deployment process

## Deployment Targets:
- Supabase (primary)
- Heroku (backup)
- Static hosting options

## Pre-deployment Checklist:
- All tests passing
- Environment variables configured
- Database migrations applied
- Assets optimized
- SEO metadata updated
