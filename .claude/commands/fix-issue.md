# Fix Issue Command

Systematic approach to debugging and fixing issues in the portfolio website.

## Steps:
1. Identify the issue scope and affected components
2. Analyze error logs and stack traces
3. Reproduce the issue if possible
4. Implement targeted fix
5. Test the solution
6. Verify no regressions

## Usage:
Type `/fix-issue` followed by issue description

## Common Issues:
- Supabase connection problems
- React component rendering issues
- TypeScript compilation errors
- Deployment configuration issues
- Form submission problems
